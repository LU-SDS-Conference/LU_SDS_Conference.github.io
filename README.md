# AIEIDA 2027 Conference Website

Static website for the International Conference on Artificial Intelligence, Embodied Intelligence and Data Analytics (AIEIDA), hosted at Lingnan University.

**Live URL (after deployment):** https://lu_sds_conference.github.io/

## Local preview

Open `index.html` in a browser, or run a local server:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080
