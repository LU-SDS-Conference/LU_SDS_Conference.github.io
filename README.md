# CRAI 2027 Conference Website

> **注意：** 这是项目说明文档，不是会议网站本身。  
> 网站首页请访问：**https://lu-sds-conference.github.io/**

Static website for the International Conference on Cybernetics, Robotics and Artificial Intelligence (CRAI), hosted at Lingnan University.

## Local preview

```bash
cd aieida-website
python3 -m http.server 8080
```

Then visit http://localhost:8080
